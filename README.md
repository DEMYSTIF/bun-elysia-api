Clone the project

```bash
git clone https://github.com/tr1sm0s1n/bun-elysia-api.git
cd bun-elysia-api
```

Install Bun

```bash
curl -fsSL https://bun.sh/install | bash
```

Install dependencies

```bash
bun install
```

Start the application

```bash
bun run dev
```

Open [browser](http://127.0.0.1:8080/swagger) for the API documentation
